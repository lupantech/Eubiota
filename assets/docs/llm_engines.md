# Supported LLM Engines

Eubiota supports a broad range of LLM engines for flexible deployment:

| Provider | Example Models | Deployment |
|----------|---------------|------------|
| **OpenAI** | gpt-4o, gpt-4-turbo, o1, o3 | Cloud API |
| **Anthropic** | claude-3.5-sonnet, claude-3-opus, claude-3-haiku | Cloud API |
| **Google** | gemini-1.5-pro, gemini-2.0-flash | Cloud API |
| **DeepSeek** | deepseek-chat, deepseek-reasoner | Cloud API |
| **Azure OpenAI** | All OpenAI models via Azure | Cloud API |
| **Dashscope** | Qwen-2.5 series models | Cloud API |
| **Together AI** | Llama, Qwen, and other open models | Cloud API |
| **vLLM** | Custom checkpoint models | Local |
| **Ollama** | Any Ollama-supported models | Local |
| **LiteLLM** | Unified interface for 100+ models | Hybrid |
